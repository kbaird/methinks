methinks
=====

An OTP library

Implements the "methinksitislikeaweasel" exercise from Richard Dawkins'
_The Blind Watchmaker_.

[![Build Status](
  https://semaphoreci.com/api/v1/kbaird/methinks/branches/master/badge.svg)]
  (https://semaphoreci.com/kbaird/methinks)

Build
-----

    $ rebar3 compile

Testing
-----

    $ rebar3 ct

or

    $ rebar3 eunit
